1. Заголовок
Недоступно поле "Количество" в заказе покупателя для услуги

2. Критичность
Значительный. Приоритет: средний

3. Окружение
Конфигурация: Демонстрационная конфигурация "Управляемое приложение" 1.0.36.2.
Файловая база, версия платформы 1С:Предприятие 8.3 (8.3.22.1709)

4. Детальное описание
В табличной части "Товары" для номенклатуры с видом "Услуга" недоступно поле "Количество". При этом в поле указано нулевое значение, но на деле оно равно 1. Это видно в общем количестве по документу. Проблема проявляется как для новых, так и для существующих документов.

5. Шаги для воспроизведения
Создать новый или открыть существующий заказ покупателя. Если документ новый, заполнить шапку. Добавить в товары номенклатуру с видом "Услуга". При этом количество не заполнится, а общее количество по документе увеличится на 1.

6. Фактический результат и ожидаемый результат
При добавлении номенклатуры с видом "Услуга" количество недоступно для редактирования. Ожидается, что количество можно отредактировать.

7. Приложение
В данном случае их нет.
